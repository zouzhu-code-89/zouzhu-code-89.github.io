查找需要安装的包

shell
(base) no.11@192 ~ % pip search <包名>

搜索报错:
ERROR: XMLRPC request failed [code: -32500]
RuntimeError: PyPI no longer supports 'pip search' (or XML-RPC search). Please use https://pypi.org/search (via a browser) instead. See https://warehouse.pypa.io/api-reference/xml-rpc.html#deprecated-methods for more information.

报错原因:
因为 pip search 服务的请求量激增，而开发新的搜索方式需要投入大量的人力物力，所以官方永久禁用XMLRPC搜索功能

解决方法: 下载 pip-search 来代替 pip search
(base) no.11@192 ~ % pip install pip-search
(base) no.11@192 ~ % pip_search django

安装 python 包

shell
pip install 
pip install <包名>==1.0.4 
pip install -r requirements.txt 
pip install <包名> -i http://pypi.mirrors.ustc.edu.cn/simple/ 
pip install -e <local project path> (这个命令相当于pip install develop)

列出已经安装的包

shell
pip list 
pip freeze

列出本地可以 editable 的项目

shell
pip list -e

列出过期的包

shell

更换镜像地址 ​

国内的pip源

  • 清华：https://pypi.tuna.tsinghua.edu.cn/simple

  • 华中理工大学：http://pypi.hustunique.com/
  • 山东理工大学：http://pypi.sdutlinux.org/
  • 豆瓣：http://pypi.douban.com/simple/

1.临时更换镜像地址

shell
pip3 install 库名 -i 镜像地址

2.windows 更换镜像地址

打开我的电脑，在地址栏中输入 %APPDATA% 按回车跳转到目标目录。在目录下创建一个pip.ini 文件。输入以下信息，以使用阿里云镜像为例。更新完毕后，再次执行下载，查看下载的地址是否更换。

注意：不管你用的是 pip3 还是 pip，方法都是一样的，都是创建 pip 文件夹。

javascript
[global]
timeout = 6000
index-url = https://mirrors.aliyun.com/pypi/simple/
trusted-host = mirrors.aliyun.com

3.Linux 或者 Mac 环境替换镜像源

创建配置文件 ~/.pip/pip.conf

shell
[root@server01 ~]# mkdir -p ~/.pip
[root@server01 ~]# touch  ~/.pip/pip.conf

在 pip.conf 配置镜像源，执行安装，查看镜像源是否替换

javascript
[global]
timeout = 6000
index-url = https://mirrors.aliyun.com/pypi/simple/
trusted-host = mirrors.aliyun.com
