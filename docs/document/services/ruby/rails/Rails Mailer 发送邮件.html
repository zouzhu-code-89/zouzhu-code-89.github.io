Rails Mailer 发送邮件 ​

Rails 使用 Action Mailer 实现发送邮件功能，邮件由邮件程序和视图控制，邮件程序继承自 ActionMailer::Base，作用与控制器类似，保存在 app/mailers 文件夹中，对应的视图保存在 app/views 文件夹中 。

生成邮件程序的步骤 ​

创建邮件程序 → 编辑邮件程序 → 创建邮件视图 → 调用邮件程序

创建邮件程序 ​

bash
$ bin/rails generate mailer UserMailer

create  app/mailers/user_mailer.rb
create  app/mailers/application_mailer.rb

create    app/views/user_mailer
create    app/views/layouts/mailer.text.erb
create    app/views/layouts/mailer.html.erb

create    test/mailers/user_mailer_test.rb
create    test/mailers/previews/user_mailer_preview.rb

下面的 code 都是创建邮件程序的时，生成的默认代码

ruby
# app/mailers/application_mailer.rb
# 所有的邮件程序都是继承 ActionMailer::Base 的基类
class ApplicationMailer < ActionMailer::Base
  # default：一个散列，该邮件程序发出邮件的默认设置，上例中，我们把 :from 邮件头设为一个值，
  #          这个类中的所有动作都会使用这个值，不过可以在具体的动作中覆盖，(继承覆盖)
  default from: "from@example.com"
  # 公共 layout 组件 -> app/views/layouts/mailer.html.erb
  #                    app/views/layouts/mailer.text.erb
  layout 'mailer'
end

# app/mailers/user_mailer.rb
# 系统默认规范，配置写在 ApplicationMailer 里，邮件类继承 ApplicationMailer 即可
class UserMailer < ApplicationMailer

end

如上所示，生成邮件程序的方法与使用其他生成器一样。邮件程序在某种程度上就是控制器。执行上述命令后，生成了一个邮件程序、一个视图文件夹和一个测试文件。

如果不想使用生成器，可以手动在 app/mailers 文件夹中新建文件，但要确保继承自 ActionMailer::Base

ruby
class MyMailer < ActionMailer::Base
end

编辑邮件程序 ​

邮件程序和控制器类似，也有称为”动作“的方法 action ，而且使用视图组织内容，控制器生成的内容，例如 html 发送给客户端；邮件程序生成的消息则通过电子邮件发送 。

app/mailers/user_mailer.rb 文件中有一个空的邮件程序，下面我们定义一个名为 welcome_email 的方法，向用户注册时填写的电子邮件地址发送一封邮件 。

下面简单说明一下这段代码，可用选项的详细说明请参见 Action Mailer 方法详解 。

  • default：一个散列，该邮件程序发出邮件的默认设置。上例中，我们把 :from 邮件头设为一个值，这个类中的所有动作都会使用这个值，不过可以在具体的动作中覆盖。
  • mail：用于发送邮件的方法，我们传入了 :to 和 :subject 邮件头。

与控制器一样，动作中定义的实例变量可以在视图中使用。

ruby
class UserMailer < ApplicationMailer
    default from: 'notifications@example.com'

    def welcome_email(user)
    @user = user
    @url  = 'http://example.com/login'
    mail(to: @user.email, subject: 'Welcome to My Awesome Site')
  end
end

创建邮件视图 ​

在 app/views/user_mailer/ 文件夹中新建文件 welcome_email.html.erb 这是视图就是邮件的模版 (UserMailer → welcome_email)，使用 xxxx.html.erb 编写 :

html
<!DOCTYPE html>
<html>
  <head>
    <meta content='text/html; charset=UTF-8' http-equiv='Content-Type' />
  </head>
  <body>
    <h1>Welcome to example.com, <%= @user.name %></h1>
    <p>
      You have successfully signed up to example.com,
      your username is: <%= @user.login %>.<br>
    </p>
    <p>
      To login to the site, just follow this link: <%= @url %>.
    </p>
    <p>Thanks for joining and have a great day!</p>
  </body>
</html>
