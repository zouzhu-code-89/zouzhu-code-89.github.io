bundle 管理项目依赖 ​

outdated 查询依赖是否过期 ​

bundle outdated 命令来查询 Gemfile.lock 文件中的包是否过时。该命令会列出 Gemfile.lock 中当前已安装的包的最新版本。

shell
ubuntu@titan-sz:~/Titan$ bundle outdated
Fetching git@github.com:MSeneadza/apartment.git
Fetching git@github.com:TimeHut/monkey-king.git
Fetching git@github.com:TimeHut/foundation.git
Fetching gem metadata from http://gems.ruby-china.com/........
Resolving dependencies...

Gem                             Current        Latest         Requested  Groups
actioncable                     7.1.2          7.1.3.3
actionmailbox                   7.1.2          7.1.3.3
actionmailer                    7.1.2          7.1.3.3
actionpack                      7.1.2          7.1.3.3
actiontext                      7.1.2          7.1.3.3
actionview                      7.1.2          7.1.3.3
activejob                       7.1.2          7.1.3.3
activemodel                     7.1.2          7.1.3.3
activerecord                    7.1.2          7.1.3.3
activerecord-postgis-adapter    9.0.1          9.0.2          >= 0       default

要检查特定的包是否过时，可以在命令后面添加包名称。例如要检查名为 "foundation" 的包是否过时，可以运行以下命令：

shell
ubuntu@titan-sz:~/Titan$ bundle outdated foundation
Fetching git@github.com:TimeHut/foundation.git
Fetching gem metadata from http://gems.ruby-china.com/........
Resolving dependencies...

Gem         Current        Latest         Requested  Groups
foundation  3.3.0 0e4ce46  3.5.2 c746977  >= 0       default
