Bundler-Audit Ruby 安全漏洞检查工具

Bundler-Audit 是一个用于 Ruby 项目的工具，主要用于对 Gemfile.lock 文件进行补丁级别的验证。它能够检查项目中使用的 gem 是否存在已知的安全漏洞，并提供相应的修复建议。Bundler-Audit 通过与 Ruby Advisory Database 进行对比，确保项目依赖的 gem 版本是安全的。
